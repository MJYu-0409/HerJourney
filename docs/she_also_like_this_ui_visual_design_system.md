# 一、整体视觉方向

## 设计主题

# Light Bloom / Midnight Bloom

采用：

- 日间柔和浅色模式（主视觉）
- 深夜沉静深色模式（情绪陪伴）

整体气质：

```text
温柔
明亮
呼吸感
低刺激
女性感
长期陪伴
高级留白
```

避免：

```text
医疗感
科技感AI
数据后台感
互联网工具感
```

---

# 二、视觉风格关键词

## UI Style

### 「Calm + Oura + 女性叙事」

- Calm

# 五、页面视觉结构

# 1. 首页 Today

## 页面目标

不是 dashboard。

而是：

# “今天有人在陪伴她。”

---

## 页面结构

### 顶部 Greeting

```text
今天也辛苦了。
```

小字：

```text
让长期变化，被温柔看见。
```

背景：

- 柔和渐变
- 慢速流动
- 类似晨雾

---

### 中部 Check-in

标题：

```text
今天感觉怎么样？
```

采用：

- emoji + pill button
- 大圆角
- 柔和阴影

选项：

- 😌 平静
- 🌙 没睡好
- 🔥 潮热
- 😞 情绪低落
- 😣 烦躁

设计重点：

# “低压力记录”

---

### AI Reflection Card

卡片风格：

- 半透明
- 毛玻璃
- 柔和投影

内容示例：

```text
最近一周，
你的睡眠正在慢慢恢复。
```

按钮：

```text
查看 Journey
```

---

# 2. Journey Timeline（核心页面）

## 页面目标

# “把长期变化变成人生故事。”

---

## 页面结构

### 顶部 Summary

```text
你已经记录了 128 天
```

下面：

- 睡眠稳定 +12%
- 情绪波动减少
- 潮热频率下降

采用：

- 柔和趋势微图
- 小型数据 Chips

---

### 中部 Trend Area Chart

图表要求：

- area chart
- 柔和渐变填充
- 不使用硬线条
- 不使用坐标轴密集标签

图表内容：

- 睡眠
- 情绪
- 压力
- 潮热

颜色：

- 雾紫
- 雾蓝
- 暖粉

---

### Narrative Timeline（核心差异化）

采用：

# 时间轴 + AI Narrative

例如：

---

## April 2025

```text
你开始频繁在凌晨醒来。
```

---

## June 2025

```text
AI 注意到：
高压力周后，
你的睡眠通常会恶化。
```

---

## August 2025

```text
你已经逐渐适应身体变化。
```

设计重点：

# “人生 Journey 感”

不是：

# “健康数据报告”

---
